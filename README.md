# 🌍 WanderPulse - Full Stack Airbnb Platform

WanderPulse is a professional, full-stack web application (like AirBnB) that allows users to discover, list, and review unique accommodations worldwide. Built with the **MVC (Model-View-Controller)** architecture, it features secure authentication, cloud-based image management, and a robust database.

## ✨ Key Features

* **Full CRUD Functionality:** Users can Create, Read, Update, and Delete listings.
* **User Authentication:** Secure Signup and Login system using **Passport.js**.
* **Cloud Image Storage:** High-quality image uploads and hosting via **Cloudinary**.
* **Interactive Reviews:** Users can leave star ratings and comments on listings.
* **Persistent Sessions:** User sessions are stored in **MongoDB Atlas** using `connect-mongo`.
* **Map Integration:** (Optional) Visualizing listing locations using Mapbox.
* **Responsive UI:** Fully mobile-friendly design built with **Bootstrap 5** and **EJS-Mate**.
* **Data Validation:** Server-side schema validation using **Joi**.



---

## 🛠️ Tech Stack

- **Frontend:** EJS (Embedded JavaScript), Bootstrap 5, CSS3, JavaScript.
- **Backend:** Node.js, Express.js.
- **Database:** MongoDB Atlas (NoSQL).
- **Authentication:** Passport.js (Local Strategy).
- **Storage:** Cloudinary API.
- **Middleware:** Method-Override, Connect-Flash, Cookie-Parser.
